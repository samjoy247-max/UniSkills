Project Overview + Roles

UniSkills একটি skill-sharing প্ল্যাটফর্ম যেখানে Student skill post দেয়, Alumni mentorship post দেয়, এবং Admin সবকিছু মডারেট করে।

Role কাজ
Student Skill পোস্ট তৈরি/এডিট, approved posts ব্রাউজ
Alumni Mentorship পোস্ট তৈরি, alumni directory ব্রাউজ
Admin Skill এবং Alumni posts approve/reject

Status Workflow

Skill Post: Create/Update → Pending → (Approved/Rejected)

Alumni Post: Create/Update → Pending → (Approved/Rejected)

Approved হলে সবাই দেখে, rejected হলে শুধু লেখক reason দেখে।

Failure/404 Examples

404 Not Found Example:
GET /accounts/skills/999999/
Result: Django default 404 page (Not Found)
404 Not Found Example:
GET /accounts/alumni/999999/
Result: alumni user না থাকলে 404
403 Forbidden: অন্যের পোস্ট edit/delete করলে

Viva Short Q/A

Q: কেন module-based API documentation দরকার?
Module অনুযায়ী ভাগ করলে exam/viva তে সহজে ব্যাখ্যা করা যায় এবং report readability বাড়ে।
Q: 404 কবে হয়?
যখন URL এ invalid post_id বা user_id দেওয়া হয়, তখন Django 404 দেয়।
Q: Admin login আলাদা কেন?
Security policy: admin login শুধুমাত্র /admin/ এ।
