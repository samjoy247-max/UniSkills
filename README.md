# UniSkills

UniSkills is a university-focused platform for peer skill sharing and alumni mentorship.

## Repository Structure

- `frontend/ux-prototype/` - Static UX prototype (HTML/CSS/JS) aligned with project requirements
- `backend/` - Django implementation workspace (next development phase)
- `docs/` - Project docs, requirements, planning artifacts

## Current Status

This repository currently includes the finalized UX prototype with role-based flow updates:

- Alumni login route and alumni-only dashboard flow
- Functional alumni career posts tab
- Alumni navigation isolated from student seeker/provider pages

## Next Step (Django)

1. Create virtual environment
2. Initialize Django project under `backend/`
3. Reuse UX screens from `frontend/ux-prototype/` as Django templates
4. Implement models and role-based auth according to SRS/JIRA
